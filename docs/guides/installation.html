📖 Guide d'Installation

Installe tes agents BantouMind AI en quelques minutes.

🔧 Prérequis

💻
Un ordinateur (Windows, Mac ou Linux)
N'importe quel PC fait l'affaire
📦
Node.js (gratuit)
Télécharge sur nodejs.org (version 16 ou +)
📱
Un numéro WhatsApp ou un site web
Pour utiliser les agents

📥 Installation Rapide

1️⃣
Télécharge le dossier
Tu reçois le dossier complet par email après paiement
2️⃣
Double-clique sur start.bat
L'installation se lance automatiquement
3️⃣
Ouvre index.html
La plateforme est prête !

⚙️ Configurer un Agent

Ouvre le fichier de configuration de l'agent (config.json ou agent.js) et modifie :

// Exemple de configuration WhatsApp Agent { "botName": "Mon Assistant", "businessName": "Ma Boutique", "contactPhone": "+225 01 02 03 04 05", "businessHours": "Lun-Sam: 8h-19h", "language": "fr" }
